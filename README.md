# 🚀 NOTAFLOW

### Smart Workspace & Mindspace Manager

> **NOTAFLOW** is an AI-powered productivity platform designed to help users organize both their **Workspace** (notes, tasks, ideas, study materials) and **Mindspace** (journals, thoughts, comfort corner, thought of the day) in one elegant application.

---

## 🌟 Project Overview

Modern productivity is not just about storing notes—it is about managing your thoughts, emotions, creativity, and daily workflow in a unified system.

**NOTAFLOW** combines:

- 📝 Intelligent note management
- 📌 Pinning important notes
- ⚡ Quick capture
- 📔 Daily journaling
- 💡 Ideas board
- 💙 Comfort Corner
- 🌍 Anonymous community thoughts
- 🤖 AI-powered summaries
- 🔍 Global search
- 👤 Profile personalization
- 🔐 Secure Firebase authentication

---

# 🧠 Core Concept

NOTAFLOW is built around two major concepts:

---

## 💼 Workspace

Workspace helps users manage everything related to productivity.

### Workspace Features

- Work Notes
- Study Notes
- Ideas Notes
- To-Do Notes
- Quick Notes
- Pinned Notes
- Global Search
- AI Summary

---

## 💙 Mindspace

Mindspace helps users manage emotions, reflections, and inspiration.

### Mindspace Features

- Daily Journal
- Comfort Corner
- Thought of the Day
- Live Community Thoughts

# 🏠 Landing Page

The landing page introduces the platform with:

- Hero section
- Feature cards
- Login / Sign Up buttons
- Start For Free option

### Guest Mode

Users can explore the app without registration.

Guest limitations:

- Maximum 3 notes
- Maximum 3 quick notes
- Limited access to advanced features

---

# 🔐 Authentication System

Powered by Firebase Authentication.

### Supported Methods

- Email & Password Registration
- Email & Password Login
- Google Sign-In
- Guest Mode

### Firestore User Document

Each authenticated user gets a document in:

```text
users/{uid}
```
